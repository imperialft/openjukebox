OpenJukebox
===========

Requirements
------------

* [VLC](http://www.videolan.org)
* [ffmpeg](http://ffmpeg.org)

Run
---

    $ git clone git://github.com/imperialft/openjukebox.git
    $ cd openjukebox
    $ bundle update
    $ rackup --server puma --port 9393

Open your browser and navigate to [http://localhost:9393](http://localhost:9393).
